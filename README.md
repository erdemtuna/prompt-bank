# Prompt Bank

A local, file based, copy only prompt library with a small React interface. Store prompts as Markdown, fill variables and optional sections, then copy the composed result into any tool. No account, no backend, no model calls.

This README is a placeholder. The full documentation, screenshots, and usage guide are prepared in a later step. See `schema.md` for the prompt file contract.
